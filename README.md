# TetOS
Kasane Teto themed Arch Linux shell

## Details
 - Supported OS: Arch Linux
 - Shell: bash
 - Window Manager: hyprland
 - Wallpaper: swww
 - App Launcher: rofi-wayland
 - Status Bar: quickshell
 - Widgets: quickshell
 - Notifications: quickshell
 - Terminal: kitty
